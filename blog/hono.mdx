## 概要

お手軽かつ無料で API サーバーを用意したかった。  
調べたら `Hono x Cloudflare workers` が良いらしいのでやってみる

## コード
https://github.com/kanemototomoki/hono-todo-api

## 出てくるライブラリとかサービスとか

- cloudflare
  - CDN サービスのひとつ。
  - cloudflare workers はサーバーレスな環境のエッジサーバーで Javascript を実行してくれるサービス
- wrangler
  - https://www.npmjs.com/package/wrangler
  - cloudflare workers のコマンドラインツール
- Hono
  - https://honojs.dev/
  - シンプルなWebフレームワーク
  - expressの上位互換？
  - 名前がカッコイイ

## Validationについて
- Hono v2.7.1からBuilt-in validatorが非推奨になってるぽい
  - https://github.com/honojs/hono/blob/main/docs/MIGRATION.md
- @honojs/validatorではなくzodを使用する
