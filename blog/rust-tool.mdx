## 概要
Rustで毎回プロジェクトに加えるツールとかをまとめる

## インストールされているツール一覧を表示する
```sh shell
rustup component list
```

## rustfix
Warningを自動で修正してくれる。
cargoに内包されている。

```sh shell
cargo fix
```

## clippy
Warningとかアンチパターンの箇所を教えてくれる。

```sh shell
rustup component add clippy

cargo clippy
```

## rustfmt
フォーマッター。
rust-analizerがフォーマットしてくれるからコマンドを叩く機会は少ないかも？

```sh shell
rustup component add rustfmt

cargo fmt
```

## cargo-watch
テストを監視しながらコーディングしたいとき

```sh shell
cargo install cargo-watch

cargo watch -x test
```
