## 概要

tokio で GET リクエストを行う

## コード

クレートの追加
```sh shell
cargo add reqwest
cargo add tokio
```

```toml cargo.toml
[dependencies]
reqwest = "0.11.13"
tokio = { version = "1.23", features = ["full"] }
```

```rs main.rs
use reqwest;

#[tokio::main]
async fn main() -> Result<(), reqwest::Error> {
    let response: String = get_input().await?;

    println!("response: {}", response);
    Ok(())
}

async fn get_input() -> Result<String, reqwest::Error> {
    let response = reqwest
        ::get("http://xxxx")
        .await?
        .text()
        .await?;

    Ok(response)
}
```
